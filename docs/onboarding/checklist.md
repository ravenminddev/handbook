# Checklist de onboarding

> Copiar y usar como checklist para cada nuevo miembro.

## Pre-arranque

- [ ] Oferta firmada y archivada
- [ ] Conflicto de intereses declarado y firmado
- [ ] Equipo físico o acceso al entorno remoto configurado
- [ ] Email corporativo creado
- [ ] Acceso a la organización de GitHub
- [ ] Acceso a canales de comunicación
- [ ] Gestor de contraseñas compartido
- [ ] Documento de seguridad de la información aceptado

## Semana 1

- [ ] Recorrido por el handbook
- [ ] Cuenta de 2FA activa
- [ ] Cifrado de disco activado
- [ ] Primer PR abierto
- [ ] 1 a 1 con cada cofundador
- [ ] Asignación de buddy

## Mes 1

- [ ] Primera entrega de valor
- [ ] Retro de onboarding
- [ ] Ajustes al proceso documentados
